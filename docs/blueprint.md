# **App Name**: Endosmanager

## Core Features:

- User Authentication: User authentication with different access levels (Attendance, Technician, Management).
- Dashboard: Dashboard displaying key metrics like open, in-progress, pending approval, and completed service orders, as well as a graph of average repair time.
- Registration Module: Client and equipment registration with detailed information and history of linked service orders.
- Service Order Management: Service Order management: creation, status updates, inspection checklists with photo uploads, budgeting, digital approval, execution tracking, and delivery confirmation.
- Reporting: Reporting module with filtering by client, equipment, status, and period, exportable in PDF/Excel format.
- Notifications via WhatsApp API: Outbound notifications via WhatsApp API for service order updates (open, approved, completed).
- AI Service Order Summarization: Use generative AI to summarize free text and checklist-style service reports. The AI tool will select which items to focus on in its summary. This feature could be integrated on the main page when reviewing each order.

## Style Guidelines:

- Primary color: Navy blue (#0C2340) for a professional and reliable look.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Light blue (#ADD8E6) to highlight key actions and elements while complementing the primary color.
- Headline font: 'Poppins', a geometric sans-serif, for a modern and clean appearance.
- Body font: 'PT Sans', a humanist sans-serif, for readability and a touch of warmth.
- Use clear and minimalist icons to represent service order status and quick actions.
- Design a clean and professional layout with rounded cards to present information clearly.
